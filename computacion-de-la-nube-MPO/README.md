# COMPUTACION DE LA NUBE (MPO)
para este trabajo voy a usar AWS como proveedor principal para poder virtualizar algunas partes de mi empresa.
sevicios de AWS utilizados: S3, Amazon RDS, EC2
S3: es para poder subir archivos a la nube y poder guardarlos sin riesgo de que se rompa un ordenador y se pierda todo. 1.20€ (50gb)
Amazon RDS: base de datos de todos mis clientes 15€
EC2: servidores virtuales y copias de seguridad de ordenadores 9€
transferencia de datos: 2€
el precio total seria de unos 25€ pero puede variar y llegar a los 45€, pero si aprovechas el free tier puedes tener el primer año gratis

# REDES LOCALES
este apartado del proyecto intermodular se centra en la asignatura redes locales, para ello he tenido que crear un esquema de una oficina.
he usado 4 ordenadores, 1 impresora, 1 router, 1 switch y 1 cloud
<img width="822" height="711" alt="image" src="https://github.com/user-attachments/assets/f1b54aef-9b51-4370-9b86-94497fcd0e3e" />
este es el esquema que he usado siendo la zona roja la zona de los empleados, la amarilla la zona del jefe con el router y el switch y por ultimo una zona para imprimir de color cian
tambien he creado una VLan para cada zona usando el gateway 192.168.10.1 para la zona de ventas, 192.168.20.1 para la zona de impresión y 192.168.30.1 para la zona del jefe
